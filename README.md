# Sales Performance Dashboard (Power BI)

## Overview
This project is a beginner-friendly Power BI dashboard created using sales data to analyze
overall business performance.

## Key Features
- Total Sales and Total Profit KPIs
- Sales by Category and Region
- Sales Trend over Time
- Top 5 Products by Sales
- Interactive Region slicer

## Tools Used
- Power BI Desktop
- Basic DAX Measures

## Key Insights
- Technology category generates the highest sales.
- West region contributes the most revenue.
- Sales increased significantly after 2015.
